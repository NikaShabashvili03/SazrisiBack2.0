üë§ User Management

1. User Login

POST /api/v1/user/login/

Authenticate a user and establish a session.

Request Body:
{ "email": "user@example.com", "password": "password123" }
Response:
  • Success (200): Returns user data and sets session cookies
  • Error (401): Invalid credentials

2. User Registration

POST /api/v1/user/register/

Register a new user account.

Request Body:
{ "email": "user@example.com", "password": "password123", "firstname": "John", "lastname": "Doe", "rePassword": "password123" }
Response:
  • Success (201): User created successfully
  • Error (400): Validation errors or email already exists

3. User Profile

GET /api/v1/user/profile/ üîí Authentication Required

Get the current authenticated user's profile information.

Response:
  • Success (200): Returns user profile data
  • Error (401): Not authenticated

üìö Categories

1. List Categories

GET /api/v2/category/list/

Retrieve a list of all available quiz categories.

Response Example:
[ { "id": 1, "name": "Category Name", "description": "Category description" } ]

2. Category Details

GET /api/v2/category/details/{category_id}

Get detailed information about a specific category.

Parameters:
  • category_id (integer): The ID of the category
Example:
GET /api/v2/category/details/1
Response:
  • Success (200): Returns category details
  • Error (404): Category not found

üß© Quizzes

1. Get All Quizzes in Category

GET /api/v2/quiz/category/{category_id}/quizzes/

Retrieve all quizzes within a specific category.

Parameters:
  • category_id (integer): The ID of the category
Example:
GET /api/v2/quiz/category/1/quizzes/

2. Quiz Details

GET /api/v2/quiz/category/{category_id}/quizzes/{quiz_id}

Get detailed information about a specific quiz.

Parameters:
  • category_id (integer): The ID of the category
  • quiz_id (integer): The ID of the quiz
Example:
GET /api/v2/quiz/category/1/quizzes/1

3. Start Quiz

POST /api/v2/quiz/category/{category_id}/quizzes/{quiz_id}/start/ üîí Authentication Required

Start a new quiz attempt.

Parameters:
  • category_id (integer): The ID of the category
  • quiz_id (integer): The ID of the quiz
Response Example:
{ "attempt_id": 2, "quiz_id": 1, "started_at": "2025-01-05T10:30:00Z" }

üìù Quiz Attempts

1. Get Attempt Questions

GET /api/v2/quiz/attempts/{attempt_id}/questions üîí Authentication Required

Retrieve all questions for a specific quiz attempt.

Parameters:
  • attempt_id (integer): The ID of the quiz attempt
Example:
GET /api/v2/quiz/attempts/2/questions

2. Submit Answer

POST /api/v2/quiz/attempts/{attempt_id}/answer üîí Authentication Required

Submit an answer for a specific question in a quiz attempt.

Request Body:
{ "question_id": 5, "selected_answer": "b", "time_taken": 5 }
Fields:
  • question_id (integer): The ID of the question being answered
  • selected_answer (string): The selected answer option (e.g., "a", "b", "c", "d")
  • time_taken (integer): Time taken to answer the question in seconds

üîê Authentication Flow

1. Register a new account using /api/v1/user/register/
2. Login using /api/v1/user/login/ to get session cookies
3. Include the session cookies in all subsequent requests
4. Verify authentication status using /api/v1/user/profile/

üéØ Quiz Flow

1. Get categories using /api/v2/category/list/
2. Select a category and get its quizzes using /api/v2/quiz/category/{id}/quizzes/
3. Start a quiz using /api/v2/quiz/category/{category_id}/quizzes/{quiz_id}/start/
4. Get questions using /api/v2/quiz/attempts/{attempt_id}/questions
5. Submit answers one by one using /api/v2/quiz/attempts/{attempt_id}/answer
